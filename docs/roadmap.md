## MVP 验收主路径

```text
进入引导式对话
  → 选择问题分类
  → 选择或填写问题
  → 选择牌阵
  → 进入独立抽牌工作区
  → 洗牌
  → 旋转牌环并逐张抽牌
  → 翻牌并落入牌位
  → 返回对话查看牌阵与照耀 / 建议摘要
  → 需要时进入详细解读页
  → 保存或分享
```

任何动画、音效或登录失败，都不应阻断这条主路径。
